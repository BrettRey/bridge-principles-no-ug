# Bridge Principles without UG

This project develops a short paper responding to Reiss's argument that Universal Grammar licenses crosslinguistic evidential travel. The productive disagreement is not that Chomskyan linguistics is unempirical, nor that crosslinguistic comparison is illegitimate. It is that the need for bridge principles does not entail that Chomskyan Universal Grammar is the only possible bridge.

Core thesis: crosslinguistic inference requires bridge principles; it does not require Chomskyan Universal Grammar. A projectibility account asks what a category lets us infer, at which explanatory level, under which contrasts, and through which stabilizers.

Public preprint: <https://lingbuzz.net/lingbuzz/010091>

Citation handle: `lingbuzz/010091`
